## 📋 Descripción del Proyecto
Este proyecto analiza el comportamiento de los clientes de una empresa de telecomunicaciones para identificar patrones de deserción (Churn). El objetivo es comprender por qué los clientes se van y proporcionar una base sólida para estrategias de retención de clientes mediante el análisis de datos y modelado predictivo.

## 📊 Hallazgos Principales
- **Servicio de Internet:** Los usuarios de fibra óptica muestran una tasa de fuga superior al 40%, lo que indica una oportunidad de mejora en la calidad o precio de este servicio específico.
- **Servicios de Seguridad:** Los clientes sin seguridad en línea tienen el doble de probabilidades de abandonar la empresa.
- **Contratos:** El modelo identifica que el tipo de contrato y los cargos mensuales son los predictores más influyentes.
